Responsive Image Gallery 

A modern, interactive Image Gallery built to showcase images efficiently. This project highlights DOM manipulation, event handling, lightbox functionality, auto-slideshow, and responsive design using HTML, CSS, and JavaScript.

✨ Features

Image Slider / Carousel – Navigate through images using next / previous buttons.

Auto-Slideshow Mode – Images change automatically every 2 seconds.

Filterable Categories – Quickly filter images by category (Chocolate, Nature, Sky).

Lightbox View – Click on any image to view in fullscreen with smooth slide + fade animations.

Slide Indicators (Dots) – Jump to any image directly using clickable dots.

Responsive Design – Works perfectly on desktop, tablet, and mobile devices.

Hover Effects & Animations – Smooth transitions and animated buttons for a modern UI.

🛠️ Technology Stack

Frontend: HTML5, CSS3, JavaScript

Frameworks / Libraries: None (Vanilla JS & CSS)

Design:

Responsive layout, hover effects, lightbox animations, Flexbox
